Hang out more with friends

With Comrado it's now easier to make fun with your friends

This application is created by Kenneth Österholm as final project in the course Web Development with Javascript in Mid Swede University 2018.

The application is created in MEAN-stack.
It's using Web Sockets for real time communicatin and image uploads is hosted on Cloudinary.
